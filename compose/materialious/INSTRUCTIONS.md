# Materialious Setup Instructions

## 1. Overview
[Materialious](https://github.com/Materialious/Materialious/blob/main/docs/DOCKER.md) is a Modern material design frontend for Invidious, focused on a clean, privacy-friendly YouTube experience. It supports local video fallback when Invidious fails and is available on Web, Desktop, Android, and Android TV.

## 2. Prerequisites
- Default port: `3000`
- `NEXTAUTH_SECRET`, `POSTGRES_PASSWORD`, and `MEILI_MASTER_KEY`, they all should be different secret phrases. The phrase should be wrapped in single or double quotes if any special characters are used.
- `NEXTAUTH_URL` should be changed to your domain name only if you are hosting it somewhere else.
