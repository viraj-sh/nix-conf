# Caddy Setup Instructions

## 1. Overview
[Caddy](https://caddyserver.com/docs/build#docker) is a powerful, extensible platform to serve your sites, services, and apps, written in Go. If you're new to Caddy, the way you serve the Web is about to change.

## 2. Prerequisites
- Create a docker network 'caddy'
- create a file named `Caddyfile` as the only file in the conf directory
- Refer this for writing Caddyfile config [https://caddyserver.com/docs/caddyfile/concepts](https://caddyserver.com/docs/caddyfile/concepts)
