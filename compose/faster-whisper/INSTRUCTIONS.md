# Faster-whisper  Setup Instructions

## 1. Overview
[Faster-whisper ](https://docs.linuxserver.io/images/docker-faster-whisper/#docker-compose-recommended-click-here-for-more-info) is a reimplementation of OpenAI's Whisper model using CTranslate2, which is a fast inference engine for Transformer models.

## 2. Prerequisites
- Default port: `10300`
