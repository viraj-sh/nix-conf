# Yaak Setup Instructions

## 1. Overview
[Yaak](https://docs.linuxserver.io/images/docker-yaak/#docker-compose-recommended-click-here-for-more-info) is a desktop API client for organizing and executing REST, GraphQL, and gRPC requests. It's built using Tauri, Rust, and ReactJS.

## 2. Prerequisites
- Default port: `3000`
