# Spotube Setup Instructions

## 1. Overview
[Spotube](https://docs.linuxserver.io/images/docker-spotube/#docker-compose-recommended-click-here-for-more-info) is an open source, cross-platform Spotify client compatible across multiple platforms utilizing Spotify's data API and YouTube, Piped.video or JioSaavn as an audio source, eliminating the need for Spotify Premium

## 2. Prerequisites
- Default port: `3000`
