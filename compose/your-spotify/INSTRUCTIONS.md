# Your_spotify Setup Instructions

## 1. Overview
[Your_spotify](https://docs.linuxserver.io/images/docker-your_spotify/#docker-compose-recommended-click-here-for-more-info) is a self-hosted application that tracks what you listen and offers you a dashboard to explore statistics about it! It's composed of a web server which polls the Spotify API every now and then and a web application on which you can explore your statistics.

## 2. Prerequisites
- Default port: `80`
