# Nexterm Setup Instructions

## 1. Overview
[Nexterm](https://docs.nexterm.dev/preview#%F0%9F%90%B3-docker) is open source server management software for SSH, VNC & RDP

## 2. Prerequisites
- Default port: `6989`
- You can generate 'ENCRYPTION_KEY' with 
    ```bash
    openssl rand -hex 32
    ```
