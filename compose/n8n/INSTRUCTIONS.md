# ConvertX Setup Instructions

## 1. Overview
[ConvertX](https://github.com/C4illin/ConvertX?tab=readme-ov-file#deployment) is a self-hosted online file converter. Supports over a thousand different formats. Written with TypeScript, Bun and Elysia.

## 2. Prerequisites
- Default port: `5678`
- Make sure `init-data.sh` file is present in the same location as `docker-composet.yaml`
