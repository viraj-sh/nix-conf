# FIleZilla Client  Setup Instructions

## 1. Overview
[FIleZilla Client](https://docs.linuxserver.io/images/docker-filezilla/#docker-compose-recommended-click-here-for-more-info) is a fast and reliable cross-platform FTP, FTPS and SFTP client with lots of useful features and an intuitive graphical user interface.

## 2. Prerequisites
- Default port: `3000`
