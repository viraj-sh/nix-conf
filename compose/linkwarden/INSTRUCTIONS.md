# Linkwarden Setup Instructions

## 1. Overview
[Linkwarden](https://docs.linkwarden.app/self-hosting/installation#docker-) is a self-hosted, open-source collaborative bookmark manager to collect, organize and archive webpages.

## 2. Prerequisites
- Default port: `3000`
- `NEXTAUTH_SECRET`, `POSTGRES_PASSWORD`, and `MEILI_MASTER_KEY`, they all should be different secret phrases. The phrase should be wrapped in single or double quotes if any special characters are used.
- `NEXTAUTH_URL` should be changed to your domain name only if you are hosting it somewhere else.
