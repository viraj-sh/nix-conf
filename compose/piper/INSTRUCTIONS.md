# Piper Setup Instructions

## 1. Overview
[Piper](https://docs.linuxserver.io/images/docker-piper/#docker-compose-recommended-click-here-for-more-info) is a fast, local neural text to speech system that sounds great and is optimized for the Raspberry Pi 4. This container provides a Wyoming protocol server for Piper.

## 2. Prerequisites
- Default port: `10200`
