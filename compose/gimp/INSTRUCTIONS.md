# GIMP Setup Instructions

## 1. Overview
[GIMP](https://docs.linuxserver.io/images/docker-gimp/#docker-compose-recommended-click-here-for-more-info) is a free and open-source raster graphics editor used for image manipulation (retouching) and image editing, free-form drawing, transcoding between different image file formats, and more specialized tasks. It is extensible by means of plugins, and scriptable.

## 2. Prerequisites
- Default port: `3000`
